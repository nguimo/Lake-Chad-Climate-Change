# Lake Chad Climate Change

## Overview

This repository contains climate change analyses over the Lake Chad Basin.

### Research Topics

- Climate variability
- Drought monitoring
- Flood risk assessment
- Extreme rainfall analysis
- Hydrometeorology
- Climate change impacts
- GIS and Remote Sensing

### Datasets

- CHIRPS precipitation
- ERA5 reanalysis
- SRTM DEM
- HydroBASINS

### Tools

- Python
- ArcGIS
- QGIS
- Google Earth Engine
- Cartopy
- GeoPandas
- Xarray

### Study Area

Lake Chad Basin (Cameroon, Chad, Niger, Nigeria, Central African Republic)
